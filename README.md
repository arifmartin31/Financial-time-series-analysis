### Overview 

This project focuses on extracting, cleaning, analyzing, and visualizing financial data for Tesla and GameStop. The objective was to combine stock price data with company revenue data and build analytical dashboards to explore trends and performance patterns.

The project demonstrates practical skills in data acquisition, web scraping, preprocessing, and visualization using Python.

### Objectives

-Extract historical stock price data using financial APIs

-Perform web scraping to collect company revenue data

-Clean and preprocess raw financial datasets

-Build visual dashboards to analyze trends

-Compare stock price movement with revenue growth

### Technologies Used

-Python

-Pandas

-yfinance API

-BeautifulSoup (Web Scraping)

-Requests

-Matplotlib

### Key Skills Demonstrated

-Financial data analysis

-API-based data extraction

-Web scraping and HTML parsing

-Data cleaning and transformation

-Time-series visualization

-Building analytical dashboards
